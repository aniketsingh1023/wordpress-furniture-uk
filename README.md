# WordPress Furniture UK - Contract Furniture Specialists

A premium WordPress theme for Contract Furniture Specialists - UK's leading banquette seating and commercial furniture provider.

## Preview
Open `preview.html` in your browser to see the full homepage.

## WordPress Installation
1. Copy the `wp-content/themes/contract-furniture-specialists` folder to your WordPress themes directory
2. Activate the theme from WordPress admin
3. Set the homepage to use the static front page

## Tech Stack
- WordPress Custom Theme
- Vanilla CSS (no frameworks)
- Vanilla JavaScript
- Google Fonts (Playfair Display, Inter, Cormorant Garamond)
- SVG icons (no icon library dependencies)
